To setup the model files, goto https://huggingface.co/Xenova/nllb-200-distilled-600M, under files-and-versions download the files manually into the structure below:

```
./models
./models/Xenova
./models/Xenova/nllb-200-distilled-600M
./models/Xenova/nllb-200-distilled-600M/tokenizer_config.json
./models/Xenova/nllb-200-distilled-600M/special_tokens_map.json
./models/Xenova/nllb-200-distilled-600M/onnx
./models/Xenova/nllb-200-distilled-600M/onnx/decoder_model_quantized.onnx
./models/Xenova/nllb-200-distilled-600M/onnx/encoder_model_quantized.onnx
./models/Xenova/nllb-200-distilled-600M/config.json
./models/Xenova/nllb-200-distilled-600M/tokenizer.json
```
